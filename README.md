## To build all packages
```bash
make all
```

## To build a specific os package
```bash
# build-oracle
# build-oracle-7
# build-oracle-8
# build-centos
# build-centos-7
# build-centos-8
# build-amzn
# build-amzn-1
# build-amzn-2
# build-ubuntu
# build-ubuntu-16.04
# build-ubuntu-18.04
# build-ubuntu-20.04
# build-debian
# build-debian-9
# build-debian-10

make centos-7
```

## To clean all old packages
`make clean-packages`

## To clean all ccache
`make clean-ccache`

## To clean all old build flags
`make clean-build-flags`
